Preferred terms

One entry per line: product names, people, acronyms, internal codenames. The dictation LLM (when enabled) uses this list to fix spelling and choose the right form when the transcript is ambiguous. Local faster-whisper also receives a short hint (weak bias only).

For reliable names and jargon, the LLM pass is the main lever; speech models cannot guarantee a custom lexicon the way a dedicated keyword spotter would.

Voice input

Dictation uses PortAudio (same stack as the rest of this app). System microphone follows the OS default input for this process; pick a specific device to lock dictation to that mic regardless of system default.

Global dictation hotkeys

macOS: global hotkeys are handled by an embedded runtime in this app process when started via ./start.sh. Hotkeys use quickmachotkey plus system registration (not a global key tap), so Input Monitoring is usually not required for hotkeys. Accessibility may still be needed for typing the transcript into other apps. If a chord never fires, check logs/hotkey-agent.log for registration errors or conflicts with another app using the same shortcut.
